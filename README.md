## Game control

|       Name       |  Key  |
|:----------------:|:-----:|
|       Move       | WASD  |
|      Pause       |   P   |
| Character Screen |   C   |
|  Confirm/Attack  | Enter |
|    Shoot/Cast    |   F   |
|      Guard       | Space |
|       Map        |   M   |
|     Mini Map     |   N   |
|     Options      |  Esc  |

## Dev control

|         Name         | Key |
|:--------------------:|:---:|
|       Load Map       |  R  |
|   Show Debug Text    |  T  |
| God Mode (never die) |  G  |
