|   Name (Main)    |  Key  |   |       Name (Dev)     | Key |

|      Pause       |   P   |   |   Show Debug Text    |  T  |
| Character Screen |   C   |   | God Mode (never die) |  G  |

|      Guard       | Space |   |                      |     |

|     Options      |  Esc  |   |                      |     |
